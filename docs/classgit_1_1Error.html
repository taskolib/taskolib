git::Error Class Reference

Detailed Description

An exception class carrying an error message.

Error is used as the standard exception class by all git-related functions throughout Taskolib. It can be used directly or inherited from.

try
{
throw git::Error("An error has occurred");
}
catch (const git::Error& e)
{
std::cerr << e.what() << "\n";
}
